GaliaJSFX_ClientAppUser

 Объектно-ориентированное представление участников бизнес-задач.
AuthRequest абстрагирует процесс аутентификации и авторизации пользователей и информационных систем (ИС).
ClientApp реализует действия ИС при взаимодействии с OAuth Provider. Например, выполняет авторизацию пользователя, выполняет получение токена доступа (Access Token), отслеживает его время жизни, использует его при вызове защищённых методов.
ClientAppUser реализует работу с учётной записью пользователя: выполнение программного входа в ИС и выхода из неё (отслеживание сеанса), получение сведений (имя, статус входа в систему, атрибуты, роли).
Обобщающий GaliaJSFX Загрузка и объединение компонент библиотеки. Выполнение цепочки логически связанных действий (Авторизация пользователя-> Получение AccessToken (например, по коду авторизации) -> Проверка валидности и обновление токена -> Вызов защищённого метода), высокоуровневая обработка ошибок.
