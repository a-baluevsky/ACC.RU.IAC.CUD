Подключение Galia JSFX к проекту

  • Подключить скрипты (заменить host на реальный адрес OAuth Provider)
    • RequireJS

      • Рекомендуется помещать скрипт RequireJS в заголовке <head>, а остальные скрипты в конце кода страницы (перед закрывающим тегом </body>) для имитации быстрой загрузки страницы:

      <script src="http://host/picketlink-oauth-provider-wwwserver/files/gfx/current/lib/require-2.1.20.js"
      data-main="http://host/picketlink-oauth-provider-wwwserver/files/gfx/current/config.js"></script>
      <script src="http://host/picketlink-oauth-provider-wwwserver/files/gfx/current/config.js"></script>
    • json2 (опционально - для браузеров без встроенной поддержки JSON)

      <script src="http://host/picketlink-oauth-provider-wwwserver/files/gfx/current/lib/json2.js"></script>

  • Добавить загрузку компонентов библиотеки с помощью RequireJS и использовать их: 

    <body onLoad="gfxDemo()">
    
      ...
    
      <DIV id=dvLog></DIV>
    
      ...
    <script language="JavaScript">
    
      function gfxDemo() {
    
      requirejs(['GaliaJSFX'],
    
      function( _gfx) {
    
      dvLog.innerHTML = 'GaliaJSFX version: ' + _gfx.version;
    
      });
    
  }
</script>

    (см. Пример подключения)
